# BetterSanPedro.ph

BetterSanPedro.ph is a volunteer-built, community-run civic portal for San Pedro, Laguna. It is part of the [Better LGU](https://lgu.bettergov.ph/GUIDE) initiative of BetterGov.ph and is independent of the City Government.

The portal is deliberately source-first: published civic information is backed by a current primary source, while unverified local content is shown as under review rather than copied from another locality.

## Current scope

- Government directory and Sangguniang Panlungsod information
- Community service categories and a PhilHealth YAKAP information hub
- Tourism, history, transparency, and detailed service content in verified-data preparation states
- PSA-backed city profile figures

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:5173`.

## Quality checks

```bash
npm run lint
npm run build
```
